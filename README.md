# Netflix Data Analysis (EDA) using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover insights about content distribution, release trends, and genre popularity.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Netflix titles dataset containing information about movies and TV shows including:
- Title
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre

## Key Analysis Performed
✔ Movies vs TV Shows distribution  
✔ Content added over the years  
✔ Top genres on Netflix  
✔ Country-wise content production  
✔ Missing data handling  

## Key Insights
- Netflix has significantly increased content after 2016
- Movies dominate the platform compared to TV shows
- The United States contributes the largest share of content

## Project Structure
- dataset → Netflix dataset
- notebook → Jupyter Notebook with analysis
- images → Visualizations generated during EDA
